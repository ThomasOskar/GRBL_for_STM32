Release Notes

Version 1.39

  • Bugfix for android 10 file picker not showing files
  • Allowed custom firmware (need to put the startup string in settings)

Version 1.28

  • Pause or Resume streaming with volume buttons.
  • Allow txt extension for gcode files
  • Jog step precision is increased to 3 decimals
  • Few bug fixed and enhancements
  • Russian translations (thanks to Mr. Николай Кирик)

Version 1.27

  • Bug Fix: Sql table not created in some mobiles
  • Bug Fix: Console window scroll is not working in low resoultion displays

Version 1.26

  • Bug Fix: App crashing on android 4.4

Version 1.25

  • Bug Fix: Ignore error 20 not working

Version 1.24

  • Added command history
  • Option to remember last gcode file location
  • Option to change baud rate for USB OTG connection
  • Option to set work coordinates for each axis directly (shortcut for G10 L20 P0)
  • Option to Ignore unsupported command errors (use this with caution)
  • Added option to change machine status report update interval (useful for slow cpu or old mobiles)
  • Increased Mpos and Wpos resolution to 3 decimals
  • Added single step mode for gcode sending
  • Dark mode for the application
  • Redesigned and organized settings page
  • Bug fixes and few UI enhancements

NOTE: This release will clear the existing application preferences.

Version 1.00

  • Additional A and B Axis Support
  • Jog pad rotation
